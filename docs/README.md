
# 前言   <!-- {docsify-ignore} -->


　　已经有博客了，为什么还要搞个文档库？依赖是这种文档式结构在检索上比博客看上去更适合查资料，如果你博客内容比较多时，你不一定记得自己原来整理过什么，文档化就是为了更快捷的查询想要的内容。同时文档化就是典型的知识分区，知识树形化，也让自己加深印象。


# 名字  <!-- {docsify-ignore} -->
